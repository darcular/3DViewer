View 3D model with xLabs!


xLabs is a lightweight head pose and gaze tracking application through your webcam. I used it as a controller in my 3D viewer so that it can response to user's head-movement.

This 3D viewer was built with WebGL using three.js, a 3D library built on top of WebGL and maintained by Mr.doob and AlteredQualia.

In order to play with head-movement, you need to install xLabs first.

What is xLabs?

xLabs is a small application for head pose and eye gaze tracking. It use the webcam; no other external devices are needed. You can find some demos here

View details »

What is the usage?

The aim of this demo is to provide a nature 3D viewer via motion-capture. You can import different formats of 3D model in to this viewer (e.g. .obj .stl .js). It was built with three.js, a 3D library built on top of WebGL.

View details »

How to play?

After installing the xLabs service and browser extension, restart your browser. You can click here to check if you have installed it correctly. In the viewer, move your head to see the nature 3D effect, and use head-rotation to rotate the object.
